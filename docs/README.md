## 프로젝트 목적

사용자가 입력한 과거 기록을 경험 단위로 정리하고, 이후 추천 단계에서 사용할 수 있는 RAG-ready vault를 구축합니다.

현재 구현은 `apps/backend`의 수집/정리/검색 파이프라인에 집중합니다.
